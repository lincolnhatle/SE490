The purpose of this README file is to explain the mechanics of this asterisk
game and the technical enhancements our team has made to it.

Rules:
The goal of the game is to reach the end exit on the right hand of the screen
without hitting any obstacles or borders along the way. As you progress in
the levels, the difficulty will increase(more obstacles between you and the
exit). Make it as far as you can!
